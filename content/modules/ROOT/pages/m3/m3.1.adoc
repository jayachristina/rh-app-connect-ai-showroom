== Setup the Lab working folder

[.indent2]
include::../_shared/close-tabs.adoc[]

. From your terminal in _Dev Spaces_, run the following command:
+
{blank}
+
[.copypaste]
****
step 4
****
+
[TIP]
--
Run the command above anytime to reset the exercise.

.Helper commands
[%collapsible]
====
* `step N` — reset the lab to the beginning of step N
* `steps` — list all available steps in the lab
* `start` or `restart` — go to the very beginning (equivalent to `step 1`)
* `ff` — fast-forward to the next step
* `rw` — rewind to the previous step
====
--
+
{empty} +

. Check your lab folder
+
After executing the commands above, have a look in your editor's tree view to confirm it looks healthy. It should be similar to:
+
--
[.indent2]

====
